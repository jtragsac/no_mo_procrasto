No Mo' Procrastinating / No Mo' Wondering

Purpose

In stressful situations people procrastinate. With this window open you will be automatically reminded not to. These buttons set timers that will allow you to only procrastinate for however long you allow yourself to.

You can use the general browsing button to set a timer without opening a browser, our quick links to access common websites, or type in the adress of sites with the other websites button.
